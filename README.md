# Зацеп имеется — сайт клуба спортивного покера (Тюмень)

Статический сайт. Без сборки — открывается как есть и заливается на любой хостинг.

## Структура
```
index.html            — лендинг (hero, турниры, форматы, рейтинг, о клубе, контакты)
account.html          — макет личного кабинета (вход + дашборд игрока)
assets/css/style.css  — все стили, бренд-токены в :root
assets/js/main.js     — хедер, меню, рейтинг-таблица, анимации
assets/img/           — сюда положить logo.svg (см. ниже)
```

## Бренд (взят из shodka.fun)

## Логотип
В шапке и футере используется векторная фишка `assets/img/logo-mark.svg`
(перерисована из фирменного знака, легко масштабируется, цвет берётся из `--accent`).

Если хочешь использовать присланные растровые рендеры:
- Полный лого с надписью → положи как `assets/img/logo-full.png` (пригодится для OG-картинки соцсетей).
- Чтобы заменить SVG-фишку на PNG, поменяй `src="assets/img/logo-mark.svg"` на свой файл.

## Бэкенд (PHP + MySQL) — авторизация через Telegram

Папка `api/` — серверная часть на PHP. Личный кабинет работает на реальных данных:
вход через Telegram Login Widget, запись на турниры, рейтинг, статистика.
Если бэкенд недоступен (например на GitHub Pages — статика), ЛК автоматически
переходит в **демо-режим** (кнопка «Войти (демо)» с примерными данными).

Эндпоинты `api/`:
```
config_public.php  — отдаёт username бота для виджета (без секретов)
auth_telegram.php  — приём данных Telegram, проверка подписи, сессия
me.php             — текущий пользователь
onboarding.php     — сохранение ника + телефона («Прежде чем начать»)
tournaments.php    — список турниров (+ записан ли игрок)
register.php       — запись/отмена на турнир
my.php             — записи, история, очки, место в рейтинге
rating.php         — рейтинг (?period=month|season|all)
logout.php         — выход
schema.sql         — схема БД + демо-турниры
```

### Запуск бэкенда на reg.ru (PHP+MySQL)
1. В панели reg.ru создай **базу MySQL** (запиши имя БД, пользователя, пароль).
2. Открой **phpMyAdmin** → выбери БД → вкладка «Импорт» → залей `api/schema.sql`.
3. Скопируй `api/config.example.php` в `api/config.php` и впиши:
   - `tg_bot_token`, `tg_bot_username` — из @BotFather;
   - `db_name`, `db_user`, `db_pass` — из шага 1;
   - `admins` — твой Telegram ID (узнать у @userinfobot), кому можно вносить результаты.
   `config.php` в git не коммитится — секреты не утекут.
4. Залей **все файлы** проекта в корень сайта (`public_html/`), включая папку `api/`.
5. В @BotFather: `/setdomain` → укажи свой домен (виджет Telegram работает только
   на привязанном домене по HTTPS, не на localhost).
6. Открой `https://<домен>/account.html` → нажми кнопку Telegram → войди.

Внесение результатов турниров (для рейтинга) пока делается напрямую в таблице
`results` через phpMyAdmin. Удобную админку можно добавить отдельным шагом.

## Домен

Боевой домен — **zacep.fun**. Он должен указывать на **хостинг reg.ru с PHP**
(где лежит `api/`), а НЕ на GitHub Pages — иначе бэкенд работать не будет.
GitHub Pages (`zamkofde-svg.github.io/zacep`) остаётся как превью/стейджинг в демо-режиме.

Подключение:
1. В reg.ru привязать домен `zacep.fun` к хостингу (Домены → Привязать к услуге),
   либо прописать DNS A-запись на IP хостинга. SSL (Let's Encrypt) включить в панели.
2. В @BotFather: `/setdomain` → `zacep.fun` (Telegram-виджет работает только на нём).
3. Проверить `https://zacep.fun/` и `https://zacep.fun/account.html`.

## Что доработать перед боевым запуском
- Реальные телефон, соцсети, ссылку на Telegram-канал и точку на карте (2ГИС или Яндекс).
- Уточнить расписание гостевых турниров (сейчас пример: вс — бар «Коробок»).
- Заполнить документы (`offer.html`, `privacy.html`, `consent.html`, `terms.html`, `rules.html`).
- Завести реальные турниры в таблице `tournaments` и убрать демо-строки.

## Деплой на reg.ru (виртуальный хостинг)
1. Зайди в панель reg.ru → «Хостинг» → твой тариф → «Файловый менеджер» (или подключись по FTP).
2. Загрузи **содержимое** этой папки (`index.html`, `account.html`, `assets/`) в корень сайта —
   обычно это `public_html/` или `www/<домен>/`.
3. Открой домен — сайт сразу работает (главная подхватывается из `index.html`).

По FTP (пример):
```
хост:   ftp.<твой-домен>
логин:  выдаётся в панели reg.ru
кладёшь файлы в:  /public_html/
```
