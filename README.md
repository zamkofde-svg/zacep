# Зацеп имеется — сайт клуба спортивного покера (Тюмень)

Статический сайт. Без сборки — открывается как есть и заливается на любой хостинг.

## Структура
```
index.html            — лендинг (hero, турниры, форматы, рейтинг, о клубе, контакты)
account.html          — макет личного кабинета (вход + дашборд игрока)
assets/css/style.css  — все стили, бренд-токены в :root
assets/js/main.js     — хедер, меню, рейтинг-таблица, анимации
assets/img/           — сюда положить logo.svg (см. ниже)
```

## Бренд (взят из shodka.fun)
- Акцент (лайм): `#8cf55b` · глубокий зелёный: `#067644`
- База: `#131316` · поверхности: `#25242a` / `#37363d`
- Текст: `#f3fdff` · приглушённый: `#63626a`
- Шрифты: Unbounded (заголовки) + Manrope (текст), подключены с Google Fonts.
  Оригинальный шрифт бренда — RF Dewi; при наличии лицензии можно заменить.

## Логотип
В шапке и футере используется векторная фишка `assets/img/logo-mark.svg`
(перерисована из фирменного знака, легко масштабируется, цвет берётся из `--accent`).

Если хочешь использовать присланные растровые рендеры:
- Полный лого с надписью → положи как `assets/img/logo-full.png` (пригодится для OG-картинки соцсетей).
- Чтобы заменить SVG-фишку на PNG, поменяй `src="assets/img/logo-mark.svg"` на свой файл.

## Что доработать перед боевым запуском
- Подставить реальные турниры/рейтинг (сейчас демо-данные в `main.js`).
- Подключить настоящий вход: Telegram Login Widget + логин/пароль (сейчас вход в ЛК — макет-переключатель).
- Реальные телефон, соцсети, ссылку на Telegram-канал и точку на карте (2ГИС или Яндекс).
- Уточнить расписание гостевых турниров (сейчас пример: вс — бар «Коробок»).

## Деплой на reg.ru (виртуальный хостинг)
1. Зайди в панель reg.ru → «Хостинг» → твой тариф → «Файловый менеджер» (или подключись по FTP).
2. Загрузи **содержимое** этой папки (`index.html`, `account.html`, `assets/`) в корень сайта —
   обычно это `public_html/` или `www/<домен>/`.
3. Открой домен — сайт сразу работает (главная подхватывается из `index.html`).

По FTP (пример):
```
хост:   ftp.<твой-домен>
логин:  выдаётся в панели reg.ru
кладёшь файлы в:  /public_html/
```
